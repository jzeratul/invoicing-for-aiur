Small invoicing system

## to start locally

### 
 - `docker-compose up` // consult docker-compose and nginx-dev.conf
 - `Run InvoicingApplication`
 - `cd reactjs-ui`
 - `npm install`
 - `npm start`
 - `localhost`  // if nginx-dev.conf is used, then port 80 is routed to 3000(UI) and 20221(API)
